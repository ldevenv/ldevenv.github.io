Games Library Application

  1. Home
  2. Games Library Application

An application to save your favorite games, write notes and the game's status.

Project information

  • Category: Android Studio Project
  • Project date: December, 2022
  • Made by: Ramón Pérez Segarra

Description

    This is a digital video game's library where you can search for the games you want, add them, write notes and save its completion state of the game.

Project features

This was an academic project to learn how to use Android Studio creating a "realistic" application that would use information from an API (custom or an existing one).

  • A functional navegation between the different activities.
  • A fragment list that holds the list of all the games.
  • Error check if you use non existing data (like searching for games that don't exist).
  • User creation and login.
  • Fetching data from a real API on runtime.
