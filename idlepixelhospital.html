Mobile Idle Game for Android

Project information

  • Category: Android game
  • Project date: May, 2022
  • Project URL: Google Play URL
  • Made by: Ramón Pérez Segarra, Carmen Martín Moreno and Mario Rodriguez Chaves.

Description

A tycoon game set in a hospital where you must earn money by treating patients, the more patients you treat the more money you will earn. Build new levels, improve your hospital and move to a new one with bigger taxes for your patients.

Project features

This was an academic project that had the purpose of studying the android game market, creating a competitive and well developed game and learn about the ASO and SEO to try and win some money by implementing the best monetization techniques for the game we were developing.

  • The game was released in Google Play.
  • Functional advertisement system with rewarded ads.
  • A functional economy that is adapted to the game's progression.
  • Ability to buy different hospitals with different money earning multipliers.
  • Ability to build and improve buildings inside your hospital to make it grow.
  • A quest system to guide the player through the game and give rewards.
  • An idle system that works while you're not playing.
  • A translation system for easily adding new languages.
