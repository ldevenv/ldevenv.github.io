Space game prototype with neonwave style

Project information

  • Category: Unity game
  • Project date: December, 2021
  • Project URL: Neonwave Feel Prototype
  • Made by: Ramón Pérez Segarra and Kai Jauregi Azcue.

Description

The game is like a mix of an old space game such as "Space Invaders"" and a rythm game like "Guitar Hero", with the main handicap that I wasn't able to make a theme for the game, so I had to go around that and just create a game that "simulated" a rythm game with the sound effects and particles.

Project features

This game was an academic project where we had to pick a previous game that we've done in the past and choose a theme for it in order to give it a lot of game feel to make it look like a more professional and final product.

  • A neonwave style.
  • Particles when hitting an enemy and for an enemy death.
  • Shooting mechanics.
  • Circular movement mechanics.
  • "Parry" mechanic with more feel when you hit an enemy "on time".

What I did

  • General design of the game.
  • Player mechanics.
  • Score system and loose condition.
  • Game feel (music, sound effects, sound effects, post-proccesses, screen shake, hit-stop).
