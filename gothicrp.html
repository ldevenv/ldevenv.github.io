Gothic Roleplay - Minecraft Server

  1. Home
  2. Gothic Roleplay - Minecraft Server

A fully customised Minecraft roleplaying server inspired by the world of the famouse RPG "Gothic".

Project information

  • Category: Minecraft custom server
  • Project date: June, 2020
  • Made by: Ramón Pérez Segarra and a friend.

Description

  • A fully customized Minecraft 1.12.2 server made with our own plugins which add a lot of new mechanics and basically redefine how Minecraft works.

Project features

This was my biggest project and fully made by 2 people just for fun because we loved roleplaying and we wanted to create something interesting and enjoyable using the world of Gothic.

  • A custom made modular professions system based on "Life is Feudal".
  • New crafting system totally different from the vanilla one.
  • New hunting system.
  • A custom hunger system with basic diseases.
  • New inventory system.
  • An automatic housing system with guest management.
  • Lockpicking system with functional locks and keys.
  • A new building system with tools and resources.
  • A map specially designed for the needs of the server.
  • And much more mechanics.

What I did

In this project I coded the plugins, designed the mechanics, managed the community, created promotional images (banners, logos, etc), writed the story and the lore of the server and did the QA.
